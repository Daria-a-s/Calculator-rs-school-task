1. Извлечение квадратного корня происходит в один клик при нажатии на кнопку
  2. Для получение отрицательного числа создана специальная кнопка. При нажатии текущее число изменить знак на противоположный
  3. При возведении числа в степень вводим: основание -> кнопка "^" -> степень
  4. Ошибка вылетает при попытке извлечь квадратный корень из отрицательного числа и при попытке возвести отрицательное число в дробную степень
